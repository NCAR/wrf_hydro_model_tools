PURPOSE

Scripts to regrid ERA5 forcing data to a WRF-Hydro domain using the ESMF
regridding functions within the NCAR Command Language (NCL). 
Note: this script is not suitable for global ERA5, it only works with regional ERA5 data.
In the ERA52WRFHydro_generate_weights.ncl file, the format of your input data is grib.
